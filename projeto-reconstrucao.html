— O que este sistema é
Um espelho, não um juiz. O objetivo não é te fazer sentir bem ou mal — é te mostrar padrões que você não vê quando está dentro do dia. Quanto mais registros você fizer, mais precisa fica a análise.
— Como usar
Registre todo dia, mesmo que seja uma linha. Venha aqui semanalmente para a revisão. Ajuste as metas conforme a realidade — não conforme o que você acha que deveria querer.
— Sobre o medo de abandonar
Você identificou isso como seu maior obstáculo. Então construímos um sistema que não colapsa se você falhar 3 dias seguidos. Uma entrada é sempre melhor que zero. O sistema é feito para aguentar recaídas — não para fingir que elas não existem.
